# 📊 Udemy Courses Dashboard Project

## 📌 Project Description
This is a data analysis project based on the Udemy courses dataset.  
It explores and visualizes course trends and provides insights using an interactive dashboard built with Streamlit.

---

## 📁 Files Included
-` udemy_dataset` → Udemy dataset
- `fds assignment 2_py` → Streamlit dashboard code
- `1st colab link.ipynb` → Analysis and answers for 26 questions
- `fds assignment1.pdf` → Final report/output file
- `dashboard link` ->Streamlit dashboard URL (ngrok)

---

## 🎯 Project Goals
- Explore and clean Udemy course dataset  
- Analyze course price, reviews, subscribers, and subjects  
- Answer 26 analytical questions  
- Build interactive dashboard for visualization  
- Generate insights for decision making  

---

## ▶️ How to Run
1. Open `fds assignment2 dashboard_py.py`
2. Install required libraries:
3.   -> pip install streamlit pandas matplotlib
4. Run the Streamlit app:
     -> streamlit run dashboard_py.py
4. Open the generated local or ngrok link

---

## 📊 Features
- Interactive filters (year, subject)
- Subscribers analysis
- Price vs subscribers visualization
- Reviews analysis
- Subject-wise course distribution
